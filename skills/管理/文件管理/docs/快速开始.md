# 快速开始

## 第一步：准备工作

### 1. 安装Git

下载地址：https://git-scm.com/download/win

安装后打开命令行，输入 `git --version` 确认安装成功。

### 2. 注册GitHub

访问 https://github.com 注册账号

### 3. 创建仓库

在GitHub上创建一个新仓库（比如叫desk）

---

## 第二步：告诉AI你的信息

在AI对话中提供以下信息：

| 信息 | 示例 |
|-----|------|
| GitHub用户名 | your-username |
| GitHub邮箱 | your@email.com |
| 仓库地址 | https://github.com/your-username/desk.git |

AI会自动帮你：
- 初始化本地仓库
- 配置用户信息
- 关联远程仓库

---

## 第三步：开始使用

### 每天三件事

| 时间 | 触发词 | 功能 |
|-----|-------|------|
| 早上 | 说"早上好" | 写今日计划 |
| 白天 | 说"你好" | 整理文件 |
| 晚上 | 说"晚安" | 写复盘 |
| 周末 | 说"周总结" | 周复盘 |

---

## 文件夹结构

```
桌面/
├── 马哲/        ← 指导思想
├── 知识库/      ← 日记、笔记
│   ├── 日记/    ← 按时间
│   └── 笔记/    ← 按主题
├── AIGC/        ← 代码项目
├── 自媒体/      ← 视频相关
└── 仓库/        ← 杂物
```
