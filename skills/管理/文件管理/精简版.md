# 个人文件管理体系 · 精简版

> 让AI帮你管理桌面文件，每天只专注三件事。

## 一句话口诀

**想写的放知识库，做的放AIGC/自媒体，杂物放仓库。**

## 文件夹结构

```
桌面/
├── 马哲/        ← 指导思想
├── 知识库/      ← 日记、笔记
│   ├── 日记/    ← 按时间（每日/周/月/季/年）
│   └── 笔记/    ← 按主题（ai编程/card/知享圈）
├── AIGC/        ← 代码项目
│   └── AI编程/
│       ├── 想法/
│       ├── work/
│       └── 已完成/
├── 自媒体/      ← 视频相关
│   ├── 素材/
│   ├── 草稿/
│   └── 成品/
└── 仓库/        ← 杂物
```

## 触发词

| 触发词 | 功能 |
|-------|------|
| 早上好 | 写今日计划（最多3件事） |
| 你好 | AI辅助整理文件 |
| 晚安 | 写每日复盘 |
| 周总结 | 写周复盘 |

## 使用前提

1. 安装Git
2. 注册GitHub
3. 创建仓库
4. **截图桌面备份**

## AI模型推荐

✅ 推荐：GLM、Deepseek、Kimi
❌ 不推荐：minimax、Qwen3
