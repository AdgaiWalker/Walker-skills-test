# Desktop File Manager

> AI-powered desktop file management. Focus on 3 things daily: Plan, Execute, Review.

## What is this?

A desktop file management system that helps you:
- 📁 Auto-classify files to correct locations
- 📝 Daily planning (max 3 tasks)
- 🔄 Daily review
- 📊 Weekly summary
- ☁️ Auto backup to GitHub

## Before You Start

⚠️ **Take a screenshot of your desktop and backup to GitHub before first use.**

## Quick Start

### Prerequisites

1. Install [Git](https://git-scm.com/)
2. Create a [GitHub](https://github.com) account
3. Create a GitHub repository
4. **Screenshot your desktop for backup**

### Usage

Say the trigger words when chatting with AI:

| Trigger | Function |
|---------|----------|
| 早上好 (Good morning) | Daily planning |
| 你好 (Hello) | File organization |
| 晚安 (Good night) | Daily review |
| 周总结 (Weekly summary) | Weekly review |

### Folder Structure

```
Desktop/
├── 马哲/ (Philosophy)    ← Guiding principles
├── 知识库/ (Knowledge)   ← Diary, notes
├── AIGC/                 ← Code projects
├── 自媒体/ (Media)       ← Video related
└── 仓库/ (Warehouse)     ← Miscellaneous
```

## Core Concept

**Notes go to Knowledge, Code goes to AIGC/Media, Misc goes to Warehouse.**
