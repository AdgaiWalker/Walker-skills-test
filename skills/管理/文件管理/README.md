# 个人文件管理体系

状态: ✅ v4.3 | License: MIT

## 为什么？

**减轻核心三件事的阻力：规划、执行、复盘。**

让你每天只专注三件事，不被文件混乱干扰。

---

## 是什么？

一个5个文件夹的目录结构：

```
桌面/
├── 马哲/        ← 指导思想
├── 知识库/      ← 日记、笔记
├── AIGC/        ← 代码项目
├── 自媒体/      ← 视频相关
└── 仓库/        ← 杂物
```

**一句话口诀：想写的放知识库，做的放AIGC/自媒体，杂物放仓库。**

---

## 怎么做？

### 功能一：人机交互整理

AI给方案，你决策，安全不丢文件。

| 触发词 | 功能 |
|-------|------|
| 早上好 | 写今日计划（最多3件事） |
| 你好 | AI辅助整理文件 |
| 晚安 | 写每日复盘 |
| 周总结 | 写周复盘 |

### 功能二：日记归档

每日日记自动归档：每日→周→月→季→年

### 功能三：仓库备份

防止文件丢失、AI误删。每次操作自动备份到GitHub。
